1-Minute Spirits Framework

Describe any spirit in 4 beats:

  • Base: grain / agave / sugarcane / neutral
  • Intensity: light / aromatic / bold
  • Character: citrus / herbal / spice / oak / smoke
  • Best use: highball / citrus cocktail / spirit-forward

If staff can say those four things, they can sell anything.
